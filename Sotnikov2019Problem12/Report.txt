Цель задачи: предсказать сорт винограда, из которого сделано вино.

Решение поставленной задачи:
1) Считывание данные в DataFrame;
2) Проверка данных на отсутствие пропусков и выделение наиболее важных
   признаков. Такими оказались Alcohol, Flavanoids, OD280/OD315, Proline;
3) С помощью pairplot выделение двух классов Flavanoids и Alcohol, используя
   которые, можно с достаточно высокой точностью разделить классы между собой;
4) Разделение выборки на тестовую и контрольную с соотношением 70:30;
5) Выбор метрик:
  a) Евклидова;
  б) Манхэттена;
  в) Чебышева.
  г) Минковского с p = (1, 5)

6) Построение графика зависимости ошибки от числа соседей.
   Были получены следующие оптимальные значения:
  а) Евклид: error = 4.93%, k = 15;
  б) Манхэттен: error = 4.93%, k = 17;
  в) Чебышев: error = 5.55%, k = 10;
  г) Для метрики Минковского описан интерактивный виджет, с помощью которого
     можно, меняя параметр p и число соседей k, посмотреть точность метода, а
     также границы между классами.

7) Для всех рассмотренных метрик из полученного графика видно, что переобучение
   наступает при k равном около 70, вследствие чего ошибка метода начинаает
   резко расти.
